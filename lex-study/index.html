lex入门

安装

在ubuntu系统上可用以下命令安装flex(flex和lex的区别这里暂且不谈)

sudo apt-get install flex

基本用法

首先创建exam.l文件,内容如下

lex的基本语法如下

lex [选项] [文件]

这里我们只需使用最简单的命令

lex exam.l

然后我们会得到一个lex.yy.c文件,对其编译并执行

gcc -o exam lex.yy.c
./exam

程序运行后可以通过键盘输入需要识别的单词,识别结果将输出在屏幕

当然,你也可以使用如下命令指定输入文件和输出文件

./exam [inputFile] [outputfile]

状态

默认状态为INITIAL
